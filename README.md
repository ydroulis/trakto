# trakto

Project of a platform where a user can signin with email and password, to access the opening view where there are all Trakto Tv modules. In this project only the "Material did√°tico" module is available, where it is shown a list of designs that were edited by the user at the Trakto account, and by clicking it opens in a new tap de design in exibition mode at the Trakto editor.

# What is used

- Angular
- Typescript
- Css

## Getting Started

First, type git clone https://github.com/ydroulis/trakto to clone the project in your directory.

Then run npm install to install alll the dependencies.

Now, run the development server:

```bash
ng serve
# or
ng serve --open
```

Open [http://localhost:4200](http://localhost:4200) with your browser to see the result.
